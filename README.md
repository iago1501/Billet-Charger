## Instalação

- To run localy you need to have yarn or npm in your machine

### Clone

- Clone the repo in the url: `https://github.com/iago1501/Billet-Charger.git`

### Setup

- After that

> Install and run with the following commands:

```shell
$ yarn
$ yarn start
```

### Stacks used

- React;
- Context API;
- MirageJS;
- Styled Components;
- Axios;
- Jest
